# Smart Study Planner (Java)

A desktop application that generates a personalized study timetable based on subjects, difficulty level, number of chapters, and days left before exams.

## Features
- GUI application built using Java Swing
- Add multiple subjects dynamically
- Chapter-wise timetable generation
- Difficulty-based scheduling
- Automatic save and load (persistent storage)
- Displays previously saved subjects on startup

## Technologies Used
- Java
- Object Oriented Programming
- Java Swing (GUI)
- File Handling

## How to Run

Compile:
javac -d bin src/planner/*.java

Run:
java -cp bin planner.Main
